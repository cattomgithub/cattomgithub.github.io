更改 Linux 的主机名(hostname)

2022-08-26 1 min read # Ubuntu # Linux

查看主机名

hostname

或者

hostnamectl

修改主机名

sudo hostnamectl set-hostname <new hostname>

这条命令会删除 /etc/hostname 文件中的主机名，然后替换为新的主机名。
