Ubuntu 刷新 DNS 缓存

2022-08-26 1 min read # Ubuntu # DNS # Linux

查看 DNS 缓存情况

sudo systemd-resolve --statistics

刷新 DNS 缓存

sudo systemd-resolve --flush-caches

刷新 Chrome 中的 DNS 缓存

在地址栏中输入 chrome://net-internals/#dns 回车打开

然后点击 Clear host cache
